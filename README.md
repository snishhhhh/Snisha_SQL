# Hotel Management Database System

## Overview

This project is a Hotel Management Database System developed using MySQL. It is designed to manage hotel operations such as guest information, room bookings, payments, housekeeping, and staff management.


## Features

Guest Management
Room Management
Reservation Management
Payment Management
Staff Management
Housekeeping
Hotel Services
Business Reports using SQL


## Tables

The database contains 13 tables:

* Hotel
* RoomType
* Room
* Guest
* Reservation
* Payment
* Invoice
* Department
* Staff
* Service
* ReservationService
* Housekeeping
* CancellationReason


## Technologies Used

* MySQL
* MySQL Workbench
* SQL



## How to Run

1. Run `schema.sql`
2. Run `sample_data.sql`
3. Run `queries.sql`
